  <implementation>
  After tests are RED (failing), implement src/execution/deno_runner.py:

  **Domain validation regex** (pre-compile at module level):
  ```python
  _DOMAIN_RE = re.compile(r'^[a-zA-Z0-9.-]+\.[a-zA-Z]{2,}$')
  ```

  **DenoRunner.execute() implementation order:**
  1. Validate each domain in allow_net_domains against _DOMAIN_RE; return ValidationFailure(invalid_domain=domain) on first failure — no subprocess spawned.
  2. Build --allow-net flag: `f"--allow-net={','.join(allow_net_domains)}"` if domains provided, else omit flag entirely (empty list = no net access).
  3. Build full command: `["deno", "run", "--no-prompt", net_flag, *extra_flags, skill_path]` (omit net_flag if domains empty).
  4. Encode params: `json.dumps(params).encode("utf-8")`.
  5. Spawn: `asyncio.create_subprocess_exec(*cmd, stdin=PIPE, stdout=PIPE, stderr=PIPE)`.
  6. Record start time: `time.monotonic()`.
  7. `asyncio.wait_for(proc.communicate(input=json_bytes), timeout=5.0)`.
  8. On `asyncio.TimeoutError`:
     - Compute `elapsed_ms = int((time.monotonic() - start) * 1000)`.
     - Kill process tree: `subprocess.run(["taskkill", "/F", "/T", "/PID", str(proc.pid)], capture_output=True)` — use `sys.platform == "win32"` check; if not win32, use `os.killpg(os.getpgid(proc.pid), signal.SIGKILL)`.
     - Drain pipes: `await proc.communicate()` (no input, just drain).
     - Return `TimeoutError(elapsed_ms=elapsed_ms)`.
  9. Non-zero exit: return `ExecutionError(exit_code=proc.returncode, stderr=stderr.decode("utf-8", errors="replace"))`.
  10. Parse stdout JSON: try `json.loads(stdout.decode("utf-8"))`; on `json.JSONDecodeError` return `ExecutionError(exit_code=0, stderr=f"stdout not valid JSON: {exc}")`.
  11. Return `ExecutionSuccess(data=data)`.

  **Critical constraints from CLAUDE.md and STATE.md:**
  - NEVER use `proc.wait()` — deadlocks when stdout > ~4KB pipe buffer.
  - NEVER use `asyncio.create_subprocess_shell` — injection risk.
  - NEVER raise exceptions for execution outcomes — always return typed result.
  - Windows: use `taskkill /F /T /PID` (os.killpg is POSIX-only, unavailable on Windows 11).
  - Always pass `--no-prompt` as first Deno flag after `run`.
  - The second `await proc.communicate()` after kill is mandatory — drains pipes, collects exit code, prevents zombie.
