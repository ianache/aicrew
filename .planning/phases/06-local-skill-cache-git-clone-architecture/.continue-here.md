### What Phase 6 replaces

**Current (HTTP fetch per file):**
- `CatalogExplorer`: fetches `catalog.yaml` + per-skill `skills.json` via httpx
- `SkillInjector._fetch_skill_md`: fetches SKILL.md via httpx
- `DenoRunner`: receives a raw.githubusercontent.com URL to execute

**Target (local git clone):**
- New `src/skill_cache.py`: manages `git clone` + `git pull` with TTL
- `CatalogExplorer`: reads `catalog.yaml` from local clone path
- `SkillInjector._fetch_skill_md`: reads SKILL.md from local path
- `DenoRunner`: receives local filesystem path to `.ts` file

### Requirements
- REPO-01: git clone to `.skills-cache/` on first use, git pull on TTL expiry
- REPO-02: zero per-file HTTP calls during routing/execution after sync
- REPO-03: GITHUB_TOKEN for authenticated clone/pull
- REPO-04: Deno executes local files, not remote URLs
