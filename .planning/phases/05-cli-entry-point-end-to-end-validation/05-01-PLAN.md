```python
import asyncio
from dotenv import load_dotenv

from src.skill_injector import SkillInjector

```

**Module-level:** `console = Console()` — one instance, reused for all output.

**`async def main() -> None`:**

1. `load_dotenv()` — FIRST statement (per CLAUDE.md: "Load with python-dotenv at entry point only"). Must be before any Config or os.environ reads.

2. `Config.from_env()` — wrapped in `try/except KeyError`: on KeyError, `console.print("[red]Error: GEMINI_API_KEY not set. Add it to .env or export it.[/red]")` then `return`.

3. Dependency wiring (in order, matching architecture layer diagram):
   ```python
   runner = DenoRunner()
   injector = SkillInjector(runner)
   explorer = CatalogExplorer(config)
   agent = CoordinatingAgent(explorer, injector, config)
   ```

4. Banner (exact text, two separate print calls):
   ```python
   console.print("AI Agents Crew v1.0")
   console.print("Type exit to quit.")
   ```

5. REPL loop — `while True:`:
   - `input("> ")` wrapped in `try/except (EOFError, KeyboardInterrupt)`: print `"\nBye."` and `break`
   - `.strip()` the result
   - If empty after strip: `continue` (silent skip, no error)
   - If `.lower() in ("exit", "quit")`: `console.print("Bye.")` and `break`
   - Otherwise: run the agent

6. Agent invocation — ALWAYS wrap in `with console.status("Thinking...") as status:`:
   ```python
   with console.status("Thinking...") as status:
       response = await agent.run(prompt, status_cb=status)
   ```
   NOTE: `console.print()` must be called AFTER the `with` block exits (pitfall 3 from research).

7. Response display — after the `with` block:
   ```python
   console.print("")   # blank line for visual separation
   ```
   Then detect error vs. success:
   ```python
   ERROR_PREFIXES = ("Skill timed out", "Skill failed", "Skill validation failed",
                     "No matching skill", "Skill domain validation")
   if any(response.startswith(p) for p in ERROR_PREFIXES):
       console.print(f"[red]{response}[/red]")
   else:
       console.print(response)
   ```

8. Entry point at module bottom:
   ```python
   if __name__ == "__main__":
       asyncio.run(main())
   ```

**Do NOT:**
- Add `Agent:` prefix to responses
- Use `sys.exit()` — use `return` from async main() for error exits
- Catch bare `Exception` to handle KeyboardInterrupt (they are different hierarchies)
- Create a new `Console()` inside the loop
- Call `console.print()` while inside the `with console.status()` block
