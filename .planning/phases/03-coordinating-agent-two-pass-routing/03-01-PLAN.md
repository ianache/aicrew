    ```python
    class Config:
    ```

    ## Step 2: Create src/agent.py — CoordinatingAgent

    Architecture decisions to implement (from RESEARCH.md):

    **Three routing paths (not two):**
    - Pass 1: Always runs — LlmAgent with output_schema=TagExtractionResult, output_key='routing', include_contents='none'. Extracts {confidence, tags}. Tag vocabulary from catalog_explorer.get_all_tags() injected into instruction.
    - High-confidence path (confidence >= threshold): Run a third lightweight LlmAgent WITHOUT output_schema or tools ('direct_answer_agent') to produce a natural language response. Do NOT try to use Pass 1 JSON output as user-facing text.
    - Low-confidence path (confidence < threshold): Call catalog_explorer.find(tags). If found: await skill_injector.build_tool(skill_def) → fresh LlmAgent with tool injected (Pass 2). If not found: return hardcoded 'no skill found' message.

    **Runner lifecycle:**
    - One Runner per CoordinatingAgent instance (expensive __init__)
    - Create fresh InMemorySessionService per run() call (prevents unbounded memory growth — Pitfall 5)
    - runner.agent is swapped via direct attribute assignment between passes — verified safe

    **Session per run():**
    - app_name = 'coordinating_agent'
    - user_id = 'default'
    - session created fresh at top of each run() call

    **Message construction:**
    ```python
    new_message = types.Content(role='user', parts=[types.Part(text=prompt)])
    ```

    **State read after Pass 1:**
    ```python
    # AFTER async for loop completes — NOT during
    updated = await session_service.get_session(app_name=..., user_id=..., session_id=...)
    routing = updated.state.get('routing', {})
    confidence = routing.get('confidence', 0.0)   # dict access, NOT routing.confidence
    tags = routing.get('tags', [])
    ```

    **JSONL log — always at end of run():**
    ```python
    import hashlib, json
    from datetime import datetime, timezone
    from pathlib import Path
    _LOG_PATH = Path('logs/routing.jsonl')

    def _write_routing_log(prompt, tags, confidence, decision, skill_name):
        record = {
            'prompt_hash': hashlib.sha256(prompt.encode('utf-8')).hexdigest()[:12],
            'tags': tags,
            'confidence': confidence,
            'decision': decision,
            'skill_name': skill_name,
            'ts': datetime.now(timezone.utc).isoformat(),
        }
        _LOG_PATH.parent.mkdir(parents=True, exist_ok=True)
        with open(_LOG_PATH, 'a', encoding='utf-8') as f:
            f.write(json.dumps(record) + '\n')
    ```

    **Final text extraction from Pass 2 events:**
    ```python
    async for event in runner.run_async(...):
        if event.is_final_response() and event.content and event.content.parts:
            text = ''.join(p.text for p in event.content.parts if p.text and not p.thought)
            if text.strip():
                final_text = text
    ```

    **Anti-patterns to avoid** (from RESEARCH.md):
    - NEVER append to agent.tools — always create fresh LlmAgent for each pass
    - NEVER use include_contents='default' — always include_contents='none' on all agents
    - NEVER read session.state['routing'] inside the async for loop — wait until loop completes
    - NEVER use global_instruction (deprecated in ADK 1.33.0) — use instruction field
    - NEVER create a new Runner per run() call — Runner.__init__ is expensive

    **CoordinatingAgent.__init__ signature:**

    ```python
            self,
            catalog_explorer,   # duck-typed: find(tags)->SkillDefinition|None, get_all_tags()->list[str]
            skill_injector: SkillInjector,
            config: Config,
        ) -> None:
    ```

    **CoordinatingAgent.run() signature:**

    ## Step 3: Write tests/test_agent.py

    Follow TDD: tests FIRST (RED), then implement until all pass (GREEN).
    Use pytest-asyncio (asyncio_mode="auto" in pyproject.toml — no @pytest.mark.asyncio needed).

    Mocking pattern for Runner.run_async():
    The runner's run_async() must be patched to yield a mock event with:
    - event.is_final_response() returning True
    - event.content.parts = [mock_part] where mock_part.text = 'answer' and mock_part.thought = False

    For session state injection:
    - Patch session_service.get_session to return a mock Session with .state = {'routing': {'confidence': X, 'tags': [...]}}
    - OR set the real InMemorySessionService to pre-populate state before Pass 1 runs

    Cleanest approach: patch 'google.adk.Runner.run_async' or patch it on the runner instance after construction. Because Runner.run_async is async and yields events, use an async generator mock:
    ```python
    async def _mock_run_async(*args, **kwargs):
        mock_event = MagicMock()
        mock_event.is_final_response.return_value = True
        mock_event.content.parts = [MagicMock(text='mock answer', thought=False)]
        yield mock_event

    runner_instance.run_async = _mock_run_async
    ```

    Inject session state for Pass 1 result by patching get_session:
    ```python
    mock_session = MagicMock()
    mock_session.state = {'routing': {'confidence': 0.85, 'tags': ['evaluation']}}
    mock_session.id = 'test-session-id'
    session_service_instance.get_session = AsyncMock(return_value=mock_session)
    session_service_instance.create_session = AsyncMock(return_value=mock_session)
    ```

    Since CoordinatingAgent creates its own Runner and InMemorySessionService internally,
    tests should construct CoordinatingAgent then access the internal _runner and _session_service
    (or use dependency injection by accepting them as optional constructor params) to inject mocks.

    RECOMMENDED: Add optional `_runner=None, _session_service=None` parameters to __init__ for testability:
    ```python
    def __init__(self, catalog_explorer, skill_injector, config, *, _runner=None, _session_service=None):
        ...
        self._session_service = _session_service or InMemorySessionService()
        self._runner = _runner or Runner(app_name='coordinating_agent', agent=self._pass1_agent, session_service=self._session_service)
    ```
    This avoids monkey-patching and keeps tests clean.
